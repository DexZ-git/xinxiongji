# 菜品成品图放这里

把每道菜的**成品图**丢进本文件夹（`assets/dishes/`），**文件名用菜名**，我会自动挂到对应菜谱当封面/详情大图。

## 格式要求
- **jpg / png / webp**（iPhone 默认的 **HEIC 网页打不开**，请在「照片」里选「拷贝/导出」为 JPG，或截图后再放）。
- 一道菜一张（成品图）。竖图横图都行，卡片会按 4:3、详情按 16:10 居中裁切。
- 大小不挑，几 MB 无所谓（本地用）。

## 说明
- 你按**菜名**丢进来就行（人好认）。**入库时我会自动把文件改成 ASCII 的菜谱 id**（如 `姜葱炒蟹.png` → `jiang-cong-chao-xie.png`）并写进 `data/recipes.json`。
  - 为什么改名：本地 Perl 预览服务器在中文 Windows 下打不开中文文件名（用系统码页、非 UTF‑8），且 ASCII 文件名对 URL / git / 跨平台都更稳。线上 Cloudflare 其实能认中文，但统一用 ASCII 最省事。
- 所以：**你放中文名 → 告诉我一声 → 我改名+接线**。app 打开即有图，不用手机里一张张传。
- 你**在手机 app 里现拍/上传**的图仍存本机 IndexedDB、覆盖内置图——两条路不冲突。
- ⚠️ **替换已有菜的图**：手机端图片是"下载一次永久缓存"（按文件名存），**换图必须换文件名**（如 `-v2.jpg`），由我在接线时处理——你还是照旧丢中文名原图即可。
