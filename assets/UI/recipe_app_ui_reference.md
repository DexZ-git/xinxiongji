# 菜谱 APP UI 视觉参考：Cozy Clean Cookbook

> 目标：可爱、有食欲、干净。可以借鉴生活模拟类 IP 的“温暖村落、小动物陪伴、手作厨房”气质，但不要复刻任何现有 IP 的角色、标识、家具、叶子 logo 或可识别元素。

## 1. 一句话方向

做一个像“软乎乎小厨房”的菜谱 App：奶油色背景、巧克力色描边、圆胖卡片、少量食材点缀；用户打开后觉得轻松、开胃、容易开始做饭，而不是进入复杂的工具软件。

## 2. 视觉关键词

软糯、温暖、干净、开胃、手作、陪伴感、生活模拟游戏感、低压力、少而精的装饰。

不追求高级餐厅感，也不追求写实美食摄影。更像一本会说话的可爱菜谱本。

## 3. 设计边界

可以参考：圆润角色、自然色、村落生活感、手作材料、暖光厨房、小图标点缀。

避免：

- 直接使用或复刻动物森友会/Nintendo 的角色、叶子标识、道具、UI 布局或可识别图形。
- 写实木纹、写实大蒜皮、玻璃拟物、复杂厨房背景。
- 大面积高饱和色、强阴影、强反光、拥挤贴纸墙。
- 过度 3D、过度渐变，导致和现有 tab 插画不搭。

## 4. 色彩系统

| 用途 | Token | Hex | 使用建议 |
|---|---:|---:|---|
| 主背景 | `cream` | `#FFF7EA` | 页面底色，替代纯白 |
| 背景色块 | `warm_blob` | `#FFF1D9` | 大面积柔和 blob、顶部氛围 |
| 卡片 | `card` | `#FFFCF4` | 内容容器，保持干净 |
| 主 CTA | `tomato` | `#FF806B` | “开始做饭”等关键按钮 |
| 辅助强调 | `butter` | `#FFD76A` | 标签、active 背景、奖励感 |
| 清爽辅助 | `mint` | `#BEEFD8` | 健康、蔬菜、轻食相关 |
| 锅具/中性色 | `sky_pan` | `#B7EEF0` | 与现有锅具蓝呼应 |
| 食材绿 | `leaf` | `#8EC66A` | 叶子、小装饰 |
| 主描边/文字 | `chocolate` | `#4C2A22` | 图标线条、重要标题 |
| 次级文字 | `warm_gray` | `#8A7A70` | 说明文字 |

颜色比例建议：奶油背景 70%，卡片白 20%，番茄/蛋黄/薄荷/叶子绿合计 10%。

## 5. 字体与排版

中文字体建议：系统圆润无衬线优先，例如 iOS 可用 PingFang SC / SF Pro Rounded 搭配；Android 可用 MiSans / Noto Sans CJK；Web 可用 `system-ui`, `Noto Sans SC`, `ui-rounded` fallback。

字号建议：

- 页面大标题：28-32sp，字重 600/700。
- 卡片标题：18-22sp，字重 600。
- 正文：15-16sp，行高 1.4-1.5。
- 标签/Tab 文案：11-13sp，字重 500/600。

文字要像朋友提醒，不像后台系统提示。例如：

- 好：`今晚吃点热乎的？`
- 好：`冰箱里有这些，够做 3 道菜`
- 避免：`数据加载失败，请重试`
- 改成：`小厨房卡住了，再试一次吧`

## 6. 圆角、描边、阴影

- 基础网格：8pt。
- 页面边距：20-24pt。
- 主卡片圆角：24-32pt。
- 按钮圆角：高度的 50%，做成 pill。
- 图标/插画描边：巧克力棕，视觉上约 2-3px。
- UI 卡片描边：浅米棕 `#E9C9A6`，1px 左右。
- 阴影：柔和、低透明度，只给主卡片和浮层。列表卡片不要都加阴影。

## 7. 页面结构建议

### 首页 / 发现

目的：减少用户选择压力，让用户马上知道“今天可以做什么”。

应包含：

- 亲切问候：`今天想吃点什么？`
- 搜索框：食材 / 菜名 / 口味。
- 快捷分类：快手菜、下饭、一人食、冰箱清空、少洗碗。
- 今日推荐主卡片：一个主菜谱 + 时间/难度标签。
- 次级推荐：食材灵感、季节菜、收藏常做。

### 菜谱详情

目的：让用户愿意点“开始做饭”。

应包含：

- 大的食物插画或温暖色块，不建议上写实食物图作为唯一视觉。
- 时间、难度、份量、口味标签。
- 食材列表：可勾选、可加入采购清单。
- 步骤：短句、编号、每步 1 个重点。
- 固定底部 CTA：`开始做饭`。

### 采购 / 冰箱

目的：把工具感做得轻松。

应包含：

- 已有食材、快过期提醒、缺少食材。
- 分类用圆胖 chip 或小卡片。
- 空状态可用小动物插画：`冰箱有点空，去加点好吃的吧`。

## 8. Tab 图标规范

已有 4 个一级 tab 图标建议继续使用透明 PNG，保持统一清晰度。

建议：

- 源图：1024px 透明底。
- 实际显示：28-34dp；active 可略大 2dp。
- Active 状态：奶油黄/浅蛋黄胶囊底，图标 100% opacity，文字巧克力棕。
- Inactive 状态：图标 75-85% opacity，文字暖灰，不加复杂阴影。
- Tab bar 背景：卡片白，圆角 28-36，浅米棕描边，可有极淡阴影。

## 9. 插画与点缀

可以使用：小云朵、蒸汽、星星、叶子、番茄、鸡蛋、胡萝卜、米粒、芝麻、小花。

装饰原则：像调味料，不像主菜。每屏 3-6 个小点缀即可，不要分散用户阅读重点。

图标/插画保持：胖圆、扁平、少量柔和高光、巧克力描边、透明底。
