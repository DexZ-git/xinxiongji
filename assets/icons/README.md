# 图标资源放这里

把图标 PNG 丢进本文件夹（`assets/icons/`），命名如下，构建时会自动引用。

## App 图标（1 个）
- `app-icon.png` —— 方形、建议 **≥ 512×512**、PNG。会由它派生 PWA 所需的 192 / 512 / apple-touch(180) 各尺寸（透明或纯底都行；iOS 主屏会自动加圆角，图案别顶到边）。

## 底部 Tab 图标（4 个）
每个 PNG、建议 **≥ 96×96**、透明底，显示时约 24–28px。命名按 Tab 从左到右：
- `tab-1.png`
- `tab-2.png`
- `tab-3.png`
- `tab-4.png`

> ⚠️ 待确认：PRD 里我先规划了 **3 个 Tab（菜谱库 / 菜单 / 更多）**，你准备了 **4 个** Tab 图标。
> 请告诉我这 4 个 Tab 分别是什么（比如 菜谱库 / 定菜单 / 备料 / 更多？），我照你的来定名并映射到 `tab-1..4.png`。

## 说明
- 原始大图（`*-source` 之类）不进构建、可 gitignore。
- 若某个图标想要"选中/未选中"两态，再告诉我，我改成 `tab-1.png` / `tab-1-active.png` 命名。
